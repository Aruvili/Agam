**Tamil Programming Language Support for Visual Studio Code**

---

## 🌟 Overview

The **Agam Language Support** extension adds **first-class editor support** for **அகம் (Agam)** — a modern **Tamil-first programming language**.

This extension makes writing Agam programs in **Visual Studio Code** simple, readable, and enjoyable.

> **Agam Language**
> Created by **Balapriyan B**
> Assisted by **Claude Sonnet 4.5**

---

## ✨ Features

* 📝 **Syntax Highlighting** for Agam (`.agam`) files
* 🇮🇳 **Tamil-first keywords support**
* 🌐 **English keyword support**
* 📄 Automatic file recognition for `.agam`
* 🎯 Clean and readable code coloring
* ⚡ Lightweight & fast

---

## 📂 Supported File Extension

```text
.agam
```

---


### Verify Installation

* Open or create a file with `.agam` extension
* Syntax highlighting should activate automatically

Example:

```agam
அச்சிடு("வணக்கம் உலகம்!")
```

## 🧩 Supported Keywords

### Tamil Keywords

```
செயல், மாறி, மாறாத, என்றால், இல்லையென்றால், இல்லை,
வரை, ஒவ்வொரு, உள்ள, திரும்பு,
அச்சிடு, உண்மை, பொய்
```

### English Keywords

```
fn, let, const, if, elif, else,
while, for, in, return,
print, true, false
```
